# 🎓 Student Grade Calculator

A Python program that calculates a student's total marks, average, percentage, grade, and pass/fail status based on marks entered by the user.

##  Features

- Accepts student name as input
- Accepts marks for multiple subjects
- Calculates total marks
- Calculates average marks
- Calculates percentage
- Assigns grades based on predefined criteria
- Displays Pass/Fail status
- Shows the final student report in a clean format

##  Technologies Used

- Python 3

##  How to Run

1. Clone this repository.
2. Open the project in VS Code or any Python IDE.
3. Run the Python file.
4. Enter the required details when prompted.
